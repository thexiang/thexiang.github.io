Configuring MAMP & Version Control for php
project
Overall:
Configuring MAMP in order to work project locally, and
using github to version control our project.
Prerequisite:
Download & install MAMP:
https://www.mamp.info/en/
Git Details:
1.Clone your php application repository into your
local.
2.Copy the folder you just cloned and put it into
Applications/MAMP/htdocs/ (you dont have to using terminal,
just drag to that folder)
3.Start server in your MAMP.
4.Once start the MAMP server, you can type the
following to connect to the database
/Applications/MAMP/Library/bin/mysql u Username p
Password
5.If you dont want always type the whole path to
connect to database, you could change it to default
directorty.
sudo nano ~/.bash_profile
alias mysql=/Applications/MAMP/Library/bin/mysql
now you can access the db using:
mysql u username p password
6.in your project make sure application/config/database.php file is correctly configured
7.Tesing it:
test your api
http://localhost:8888/Orange/index.php/api/item_class/items
